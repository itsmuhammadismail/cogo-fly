{% blocktrans %}This brand is the result of a worldwide issue: There are far too many people who travel alone!{% endblocktrans %}

{% trans "The concept" %}

{% trans "The idea came to light after several professional and private trips I have been able to do over the years..." %}

{% trans "This social network website is a worldwide concept which will help to avoid travelling, spending a weekend, make a business trip, do some rides (cultural, sport, education, etc...), or just going on an outing, ALONE...if someone just feel ALONE, as well and for any reasons, this website is then the good one !" %}

{% trans "This situation, unfortunately, doesn't have \"borders\"...It can touch a person, a group of person, couples, families and also all socio-professional categories." %}

{% trans "Cogofly's primary goal is to help people to avoid travelling alone any more. The second one is simply to help them to no longer be alone." %}

{% trans "It also naturally deals with the following notion: 'Who has never felt like they're always well surrounded, but ended up feeling rather alone ?'" %}

{% trans "The brand: name + logo" %}

{% trans "It was of paramount importance to find a site name which was both catchy and easy to remember for all members, plus anyone else reading this page (verbally and visually)." %}

{% trans "It was therefore absolutely necessary to find:" %}

  • {% trans "A short name, easy to pronounce and remember (Facebook, Google, Twitter, Airbnb, Viadeo, etc...)." %}

  • {% trans "A simple, clear and visual logo." %}
